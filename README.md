# Zomato Data Analysis

## Overview

This project focuses on analyzing the Zomato restaurant dataset using Python in Google Colab. The analysis explores restaurant ratings, cuisines, pricing, and customer preferences to uncover meaningful insights.

## Tools and Technologies

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib

## Dataset

The project uses the Zomato Restaurant Dataset for exploratory data analysis and visualization.

## Objectives

* Analyze restaurant ratings and reviews
* Identify popular cuisines
* Study price ranges across restaurants
* Explore trends and patterns in restaurant data

## Project Files

* Zomato Analysis.ipynb – Google Colab notebook containing the complete analysis

## Conclusion

The project demonstrates data cleaning, exploratory data analysis, and visualization techniques to derive insights from real-world restaurant data.
